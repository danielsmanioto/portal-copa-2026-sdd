# [05] Construir páginas de seleção (`selecao/{slug}.html`)

## Objetivo
Implementar a página de detalhe por seleção com elenco completo.

## Entregáveis
- Template/página para seleção baseada em `data/teams/{slug}.json`.
- Renderização de jogadores com nome, idade, clube, posição e foto.
- Cálculo de idade a partir de `birthdate`.

## Dependências
- Conclusão das tarefas [02], [03] e [04].
