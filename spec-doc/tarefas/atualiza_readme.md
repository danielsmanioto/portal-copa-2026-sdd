Importante:

* Não invente tecnologias que não existam no projeto
* Baseie-se apenas no código e estrutura real encontrados
* Caso algo não exista, omita elegantemente
* Caso encontre múltiplos serviços, documente todos
* Caso exista Docker Compose, documente os containers
* Caso exista CI/CD, documente pipeline encontrado
